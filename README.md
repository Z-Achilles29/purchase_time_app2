# 🛍️ Customer Purchase Time Prediction

## 📌 Project Overview

Customer Purchase Time Prediction is a Machine Learning project that predicts when a customer is most likely to make their next purchase based on historical transaction data and customer behavior. The project helps businesses improve customer retention, optimize marketing campaigns, and personalize promotional offers.

The complete workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, and prediction.

---

## 🎯 Objectives

* Predict the customer's next purchase time.
* Analyze customer purchasing behavior.
* Build an accurate machine learning prediction model.
* Compare different regression algorithms.
* Generate business insights from customer transaction data.

---
## 📂 Dataset

The dataset contains customer transaction records with features such as:

* Customer ID
* Purchase Date
* Order Amount
* Product Category
* Quantity
* Payment Method
* Customer Location
* Previous Purchase History
